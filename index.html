About me

  • I completed my PhD in biochemistry at the University of Stellenbosch, South Africa, in January 2024, focussing on both computational and experimental biochemistry. My research has been focused on the dynamical systems in nature, specifically cyclical ones, such as the metabolic oacillatins in yeast cell and the calcium oscillations and insulin secretion from pancreatic β-cells.


    I make use of both wet lab techniques and computational biology for my research by creating mathematical models of biological phenomena and validating these models with biological data. With expertise in assay development and enzyme kinetics I am able to create detailed kinetic models which can be used as tools for further research, including effective drug target discovery.

Projects

  • Pancratic β-cell metabolism and synchronisation - Pancreatic β-cell metabolism forms a crutial step in linking cellular metabolism to insulin secretion. The synchronisation of insulin secretion from individual beta cells is also of vital importance to heathly pancreas function.

  • Yeast glycolytic oscillations - Under certain conditions glycolysis shows oscillatory behaviour which is largely controlled by the allosterically controlled enzyme phosphofructokinase. Using yeast as a model organism we are able to further understand the fundamental natures of glucose metabolism.

Tools

  • Cell Detection - This tool identifies individual cells in an image taken using fluorescent microscopy. The image is segmented so that the intensities for each cell can be attained and used for further analysis. This tool can be applied to single images for a signle intensity value for each cell or can be used in a time series to get the intensity over time for each identified cell.

  • Antimicrobial Peptide Finder - A web app that can be used to screen for antimicrobial peptide activity in protein sequences. The web app alows the user to upload a .fasta file of protein sequences which the program will make predictions on. The program makes predictions based on a combination of a random forest classifier, gradient boost classier, dense neural network, recursive neural network and convolutional neural network all trained on available data on existing antimicrobial peptides.
